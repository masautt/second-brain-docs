# Future Apps and Roadmap

## minapp

A planned third pillar for self-tracking data that does not fit finance or people.

Potential domains:

- Spotify listening stats
- Personal metrics
- Miscellaneous tracked datasets

## ETL Evolution

`finapp-etl` is expected to evolve into `second-brain-etl` over time.

| Phase | Scope |
|-------|-------|
| Now | finapp data only |
| Next | pinapp data management |
| Future | minapp and additional data sources |
